 🚀 Projeto em destaque: Sistema de Chamados (Help Desk)
 
Sistema web completo para abertura, acompanhamento e gestão de chamados de suporte. Ideal para times que querem organizar demandas internas sem depender de planilhas ou e-mails.

 Funcionalidades
-  Autenticação de usuários com perfis de acesso (Admin / Usuário)
-  Abertura, edição e acompanhamento de chamados
-  Painel administrativo com visão geral
  - Layout responsivo e intuitivo

🛠️ Tecnologias utilizadas
- **C#** - Lógica e regras de negócio
- **ASP.NET Core MVC** - Arquitetura da aplicação
- **Entity Framework** - Acesso e mapeamento dos dados
- **HTML5 / CSS3** - Interface e estilização
- **SQL Server / MySQL** (Banco de dados)
