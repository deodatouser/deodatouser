# 🚀 Projeto em destaque — Sistema de Chamados (Help Desk)

Sistema web completo para abertura, acompanhamento e gerenciamento de chamados de suporte técnico.

Ideal para empresas e equipes que desejam organizar demandas internas sem depender de planilhas ou e-mails.

## ✨ Funcionalidades

- 🔐 Autenticação de usuários
- 👤 Controle de acesso (Admin / Usuário)
- 📌 Abertura e gerenciamento de chamados
- 🛠️ Edição e acompanhamento de solicitações
- 📊 Painel administrativo
- 📱 Layout responsivo e intuitivo
